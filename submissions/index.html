The Waterloo Mathematics Review

Submissions


The Waterloo Mathematics Review is currently accepting submissions. The deadline for consideration for inclusion in Volume III, Issue 1 is June 1, 2017.

Articles

The Review invites both original research and quality expository articles written for an advanced undergraduate audience on topics related to mathematics at the undergraduate level including: computer science, mathematical physics, statistics, mathematical finance, and actuarial science. The Review accepts submissions via electronic or letter mail. If submitting by e-mail, please submit an Adobe PDF, as well as the source files (figures, TeX files, Microsoft Word documents, etc.), as attachments to submissions@mathreview.uwaterloo.ca. TeX or LaTeX is strongly preferred.

Authors are invited to use the Review's in-house LaTeX style to facilitate typesetting of submissions.

The Review requires submissions adhere to the following guidelines:
  1. Length: Submissions should be between 3 and 20 pages in length. Longer submissions may be accepted in extreme cases, please contact the editor separately with such an enquiry.

  2. Citations: A list of references must accompany a submission. BibTeX is strongly preferred as the citation format.

  3. Abstract: A concise abstract must accompany a submission.

Mathematical Art

The Review invites graphics of a mathematical nature. Submissions of graphics should include both black-and-white and colour versions in production quality, along with a description of the mathematics being depicted and a few words on how the image was produced. Source code submission is not required when images are produced using a custom program, however, should an artist request it the Review will publish the source code on our website, to accompany the electronic version of the issue. Due to the need for production quality digital formats, art is only accepted via electronic mail. Please submit all relevant files as an attachment to submissions@mathreview.uwaterloo.ca.
